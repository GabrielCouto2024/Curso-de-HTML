Tipos de caixa em Css

Caixas Box-Level (block)

São caixas que ocupam uma linha inteira. Ex:

  • Tags h1
  • Tags p
  • Divs

Caixas inline-level (inline)

São caixas que ocupam apenas o espaço necessario. Ex:

  • Links
  • Tags span
  • inputs ou buttons

Partes das caixas

  • Espaço do conteudo (pode ser modificado por height e width)
  • padding
  • border
  • outline
  • margin
