Como a internet funciona

Como os dados são representados em um computador

Os dados são representados em um computador através de bits, que são unidades binárias de 0s e 1s.

Grupos de 8 bits formam bytes, que por sua vez representam caracteres, números e outras informações. Diferentes tipos de dados (como inteiros, floats, texto)

Ex: 01000010 = A

têm formatos específicos de representação em binário, seguindo convenções definidas pelos sistemas de codificação, como ASCII ou Unicode.

Multiplos do byte

Os múltiplos do byte são geralmente expressos em termos de potências de 2. Ex:

  • Kilobyte (KB): 1 kilobyte = 1024 bytes;
  • Megabyte (MB): 1 megabyte = 1024 kilobytes;
  • Gigabyte (GB): 1 gigabyte = 1024 megabytes;
  • Terabyte (TB): 1 terabyte = 1024 gigabytes;
  • Petabyte (PB): 1 petabyte = 1024 terabytes;

Diferença entre megabits e megabytes

Megabits (Mb) e megabytes (MB) são unidades de medida de dados, mas representam coisas diferentes. Um megabit é uma unidade de medida de taxa de transferência de dados, enquanto um megabyte é uma unidade de medida de armazenamento de dados.

Megabit (Mb): Geralmente usado para medir a velocidade de transferência de dados em redes ou a largura de banda da conexão à internet. 1 megabit é igual a 1 milhão de bits.

Megabyte (MB): Usado para medir a capacidade de armazenamento de dados em dispositivos como discos rígidos, pendrives ou a quantidade de dados em um arquivo. 1 megabyte é igual a 8 megabits.

Portanto, ao falar sobre velocidade de internet, é comum usar megabits por segundo (Mbps), enquanto em armazenamento de dados, como o tamanho de arquivos, utiliza-se megabytes (MB).

O que é cliente e servidor

Cliente:

- Definição: Um cliente é um dispositivo, programa ou aplicativo que solicita serviços ou recursos de outro sistema, conhecido como servidor.
- Função: Recebe e exibe informações fornecidas pelo servidor, processando dados localmente quando necessário.
- Exemplo: Um navegador da web é um cliente que solicita páginas da web a servidores para exibi-las ao usuário.

Servidor:

- Definição: Um servidor é um sistema que fornece serviços, recursos ou dados a clientes, respondendo às solicitações recebidas.
- Função: Gerencia e distribui recursos, atendendo às demandas dos clientes e garantindo a integridade e segurança dos dados.
- Exemplo: Um servidor de e-mail armazena, gerencia e entrega mensagens para os clientes de e-mail dos usuários.

O que é o modem?

Um modem é como um tradutor para computadores. Ele transforma as informações do seu computador em sinais que podem viajar pelos cabos ou linhas telefônicas, e também traduz os sinais que recebe de volta para que o computador consiga entender. É como se fosse um intérprete que facilita a comunicação entre o seu computador e a internet.
